# Retail-POS-Profitability-Analysis

This project analyzes a retail store's Point-of-Sale (POS) and inventory data to identify the root causes of declining profitability after increased inventory purchases.

🎯 Business Problem
Despite stable sales, the business experienced a decline in profit after expanding inventory purchases. The goal was to identify key drivers of loss and recommend data-driven solutions.

🛠 Tools & Technologies
- SQL (Data Analysis)
- Python (Pandas, Matplotlib, Seaborn)
- Tableau Public (Dashboard Visualization)

📁 Dataset
Synthetic retail dataset including:
- Sales transactions
- Inventory purchases
- Discounts
- Returns
- Shrinkage

🔍 Key Analysis Performed
- Profit trend analysis
- Inventory vs sales comparison
- Category-level profitability
- Discount impact analysis
- Return and shrinkage evaluation

📈 Key Insights
- Overstocking led to excess inventory and reduced cash flow
- High discount rates significantly reduced profit margins
- Certain product categories showed high return rates
- Supplier cost differences impacted overall profitability
- Inventory shrinkage contributed to hidden losses

💡 Business Recommendations
- Implement demand-based inventory planning
- Optimize discount strategies
- Improve supplier selection and negotiation
- Reduce return rates through quality control
- Strengthen inventory monitoring systems
